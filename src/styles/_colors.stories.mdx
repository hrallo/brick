utility classes for `background`, `color`, and `fill`.

## Class Usage

### Background

Background utilitiy classes are available for each color as
`background-[palette]-[shade]`. For example:

<div class='background-color-blue-400'>
  This div has a background color of $color-blue-400.
</div>
<Source
  language='html'
  code={
    '<div class="background-color-blue-400">This div has a background color of $color-blue-400.</div>'
  }
/>

### Color

Color utilitiy classes are available for each color as
`color-[palette]-[shade]`. For example:

<div class='color-pink-500'>This div has a text color of $color-pink-500</div>
<Source
  language='html'
  code={
    '<div class="color-pink-500">This div has a text color of $color-pink-500</div>'
  }
/>
